Principais Tags HTML5 e CSS3

CSS3 Principais Tags

  1. Background: cor de fundo, imagem, transparência.
  2. Background-attachment: rolagem do fundo / marca d’água.
  3. Background-image:imagem de fundo.
  4. Background-color: cor de fundo ou transparência.
  5. Border: largura, estilo e cor de todas as 4 bordas.
  6. Border-bottom: largura, estilo e cor da borda inferior.
  7. Border-bottom-style: estilo da citada borda.
  8. Border-color: cor das 4 bordas.
  9. Border-style: estilo de todas as 4 boras.
  10. Clear: elementos flutuantes à esquerda ou à direita de um elemento.
  11. Color: cor de primeiro plano.
  12. Float: se o elemento flutua.
  13. Font: estilo, variante, peso, tamanho e altura da linha do tipo de fonte.
  14. Font-family: tipo de fonte.
  15. Font-size: tamanho da fonte.
  16. Font-style: fonte itálico.
  17. Fonte-variant: fonte bold.
  18. Font-weight: peso da fonte de claro a negrito.
  19. Height: altura exibida ao elemento.
  20. Width: largura exibida ao elemento.
  21. Margin: tamanho de todas as 4 margens.
  22. Padding: espaço em torno de um elemento em todos os lados.
  23. Text-align: alinhamento do texto.
  24. Text-transform: transformação para todas maiúsculas, minúsculas ou inicial maiúscula.
  25. Vertical-align: alinhamento vertical do elemento.

HTML5 Principais Tags

  1. DOCTYPE: define o tipo de documento
  2. Html: define o início do documento html, todos as demais tags devem estar dentro desta tag
  3. Head: cabeçalho onde ficar os metadados do documento, como metatgs, css e javascript
  4. Ttitle: define o título do documento, mostrado na barra de títulos do navegador e como link nos mecanismos de busca
  5. Base: define a url base do documento
  6. Link: utilizado para linkar um arquivo de imagem ou css a página html
  7. Script: utilizado para escrever ou anexar um arquivo normalmente javascript a página
  8. Meta: metatags, como charset, description, viewport ou keywords
  9. Style: utilizada para adicionar um código css a página
  10. Body: corpo ou área principal do documento, dentro deste deve ficar o conteúdo da sua página
  11. Header: define um cabeçalho da página ou de uma seção, normalmente pode conter um menu (nav), um logo ou link
  12. Nav: define uma área de navegação contendo links, para a formação de menus com hiperlinks
  13. Footer: define o rodapé de uma página ou seção
  14. Main: define o conteúdo principal da página, deve ser utilizado apenas uma única vez em cada página
  15. Section: define uma seção do documento
  16. Article: pode ser um post ou notícia de um fórum ou blog, define um conteúdo independente
  17. H1 a H6: representam 6 níveis de títulos, sendo o h1 de maior importância
  18. P: define um parágrafo
  19. Br: define uma quebra de linha
  20. Div: define uma camada sem formatação, genérico
  21. Ul: define uma lista não ordenada
  22. Li: define um elemento ou item da lista não ordenada
  23. A: define um link que deve ser apontado com o atributo href
  24. Img: define a utilização de uma imagem (JPG, GIF, PNG e outras). A imagem deve ser apontada com o uso do atributo src
  25. Strong: apesar de deixar em negrito, representa a importância deste texto no meio da frase
